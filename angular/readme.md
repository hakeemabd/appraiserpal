## INSTALLATION 
### Execute this
```
    cd ./angular 
    npm install  
    bower install 
```
## Fix Bootstrap Less
In /bower_components/bootstrap/less/variables.less replace the line #35 to `@link-hover-color:      darken(#337ab7, 15%);`
### Execute this
```
    gulp build
```

## Start local server
```
    gulp serve
```

## Don't forget install
```
    npm install -g less
    npm install -g gulp
```
